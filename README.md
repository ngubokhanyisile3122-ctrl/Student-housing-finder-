# Student Residence Finder

## Project Description
Finding reliable off-campus housing is a major challenge for students. This project is a centralised web application designed to help university students browse, compare, and secure nearby accommodations. By consolidating residence information into one accessible platform, we aim to eliminate the need for students to rely on scattered word-of-mouth information.

## Tech Stack (Planned)
* **Frontend:** React.js / Tailwind CSS
* **Backend:** Node.js with Express
* **Database:** MongoDB
* **Design & Planning:** Figma, Trello
